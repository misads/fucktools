# fucktools
一些(功能实用的)小工具，在运气好的情况下不会需要用到。

| 文件               | 用途                                                         |
| ------------------ | ------------------------------------------------------------ |
| fuck_class.py      | 一个选课脚本，适用于2020年版本的tju系统选课。                |
| net_reconnecter.py | 一个网络掉线自动重连脚本(需要sudo权限)，它可以确保远程的设备不会失去连接。在重新连接时会将ip地址发送到指定服务器。 |
